# Frontend — Plataforma de Tutorías

SPA en React + Vite.

Documentación general del proyecto: [../README.md](../README.md).

## Inicio rápido

```bash
pnpm install
cp .env.example .env
pnpm dev
```

App en `http://localhost:5173`. Requiere la API en `http://localhost:4000/api`.

## Build

```bash
pnpm build
pnpm preview
```
